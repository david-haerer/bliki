# Philosophie

## Was ist der naturalistische Fehlschluss, bzw. Humes Gesetz?

- Von einem Ist-Zustand kann nicht auf einen Soll-Zustand geschlossen werden.
- Nur weil etwas so ist, heißt das nicht, dass es auch so sein soll.

## Was ist der moralistische Fehlschluss?

- Von einem Soll-Zustand kann nicht auf einen Ist-Zustand geschlossen werden.
- Nur weil etwas so sein soll, heißt das nicht, dass es auch so ist.

## Was besagt der kategorische Imperativ von Immanuel Kant?

- Handle nur nach derjenigen Maxime, durch die du zugleich wollen kannst, dass sie ein allgemeines Gesetz werde.
- Eine Maxime ist die Art und Weise, wie man handelt.
- Ein Zweck ist ein Ziel.
- Benutze Menschen nicht bloß als Mittel zum Zweck sondern betrachte sie stets auch als Selbstzweck.

## Was sind die mittleren Prinzipien von Barbara Hermans?

- Anti-Manipulations-Prinzip: Andere dürfen nicht durcgh versteckte Informationen, emotionale Ausnutzung oder Täuschung zu Handlungen gebracht werden, die sie bei voller Einsicht ablehnen würden.
- Anti-Zwangs-Prinzip: Pysische oder psychische Gewaltanwendung zur Bestimmung fremder Willensentscheidungen verletzt die Selbstgesetzgebung.
- Respekt-Erkennungs-Prinzip: Jeder Mensch muss als gleichberechtigter Ko-Gesetzgeber anerkannt werden, unabhängig von Nutzenkalkulationen.
- Prozedurale Fairness: Regeln müssen so beschaffen sein, dass sie von allen Betroffenen als vernünftig akzeptiert werden könnten.
- Anti-Parternalismus-Prinzip: Fremdbestimmung "zu meinem Besten" untergräbt meine Würde als selbstbestimmtes Wesen.
